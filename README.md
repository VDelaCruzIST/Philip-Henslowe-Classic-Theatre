# Philip Henslowe Classic Theatre Website

## Overview
This repository contains a website created as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website promotes a fictional theatre company and showcases upcoming productions. The project demonstrates web development concepts including page layout, navigation design, typography, and custom font integration using HTML5 and CSS3.

## Features
* Multi-page website structure
* Navigation menu system
* Custom font implementation using @font-face
* Styled theatre production listings
* Semantic HTML5 elements
* Responsive page layout
* External font and image asset management

## Technologies Used
* HTML5
* CSS3
* Custom Web Fonts (TTF and WOFF)

## Project Files
* `ph_plays.html` – Main theatre productions page
* `ph_layout.css` – Layout and page structure styling
* `ph_styles.css` – Typography and visual styling
* Custom font files (`.ttf`, `.woff`)
* Theatre logo and image assets

## Skills Demonstrated
* Front-end web development
* CSS layout design
* Typography and custom font integration
* Website navigation
* File organization and asset management
* Semantic HTML structure

## Academic Project
This project was completed as part of web development coursework and demonstrates foundational front-end development skills using HTML and CSS.
